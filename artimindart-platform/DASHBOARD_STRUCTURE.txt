ENVIRONMENT VARIABLES:
======================

Required in .env.local:
  NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
  NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here

How to get these:
  1. Create project at https://supabase.com
  2. Go to Project Settings > API
  3. Copy "Project URL" and "anon" key


DATABASE SCHEMA:
================

Table: projects
  ├─ id (UUID, Primary Key)
  ├─ user_id (UUID, Foreign Key → auth.users)
  ├─ name (VARCHAR(255), Required)
  ├─ description (TEXT, Optional)
  ├─ status (VARCHAR(50), Default: 'active')
  │  └─ Options: 'active' | 'completed' | 'archived'
  ├─ created_at (TIMESTAMP, Default: NOW())
  └─ updated_at (TIMESTAMP, Default: NOW())

Indexes:
  ├─ idx_projects_user_id (on user_id)
  └─ idx_projects_created_at (on created_at DESC)

Row Level Security Policies:
  ├─ SELECT: Users see only their own projects
  ├─ INSERT: Users insert only their own projects
  ├─ UPDATE: Users update only their own projects
  └─ DELETE: Users delete only their own projects


ROUTES:
=======

Public Routes:
  GET  /                     - Home page
  GET  /auth/login           - Login page
  GET  /auth/signup          - Sign up page
  POST /auth/login           - Login API (Supabase)
  POST /auth/signup          - Sign up API (Supabase)

Protected Routes (require authentication):
  GET  /dashboard            - Main dashboard
  GET  /projects             - Projects page

API Calls (via Supabase client):
  GET    /projects           - Fetch user's projects
  POST   /projects           - Create new project
  PATCH  /projects/:id       - Update project
  DELETE /projects/:id       - Delete project


QUICK START COMMANDS:
=====================

1. Setup Supabase credentials:
   cp .env.local.example .env.local
   # Edit .env.local with your Supabase URL and Key

2. Install dependencies:
   npm install
   # (Already done - dependencies are in package.json)

3. Run SQL in Supabase:
   # Copy SETUP_SUPABASE.sql to Supabase > SQL Editor
   # Execute to create projects table

4. Start development server:
   npm run dev
   # Open http://localhost:3000

5. Test the app:
   - Sign up: /auth/signup
   - Login: /auth/login
   - Dashboard: /dashboard
